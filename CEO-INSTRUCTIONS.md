# Инструкция — сайт Музея русской сказки на вашем компьютере

## Первый запуск (один раз)

1. Создайте папку `C:\projects\Muzei-skazki`.
2. Положите в неё 4 файла, которые прислал Павел: `install.bat`, `pull.bat`, `push.bat`, `.env`.
3. Запустите `install.bat` двойным щелчком и дождитесь надписи **DONE**.
   Он сам установит всё необходимое: Git, Node.js, код сайта с GitHub, базу данных
   и помощника Claude Code. Если он попросит закрыть окно и запустить себя ещё раз —
   так и сделайте (это нормально после установки Git или Node).

## Ежедневная работа

1. **`pull.bat`** — забрать свежую версию сайта с GitHub (всегда начинайте с этого).
2. **`start.bat`** — запустить сайт. Откройте в браузере **http://localhost:3100**.
   Пока это окно открыто — сайт работает. Закрыли окно — сайт остановился.
3. Работайте с сайтом и админкой (http://localhost:3100/login).
   Чтобы менять сайт с помощью ИИ: откройте терминал в папке сайта
   (в Проводнике в адресной строке наберите `cmd` и Enter), затем наберите `claude`.
   Дальше просто пишите по-русски, что нужно изменить.
4. **`push.bat`** — когда закончили, отправьте изменения на GitHub.
   Программа спросит короткое описание («что я сделала») и сама поднимет
   версию сайта (V01.001 → V01.002 → …). Версия видна внизу каждой страницы.

## Важно знать

- Ваши настройки (`.env`), база данных и загруженные картинки **никогда не стираются**
  при обновлениях — они не хранятся в GitHub.
- Если `pull.bat` предупреждает про несохранённые изменения — сначала запустите
  `push.bat`, потом снова `pull.bat`.
- Если что-то пошло не так — сфотографируйте экран и отправьте Павлу,
  или спросите Claude Code: наберите `claude` и опишите проблему.
